RetroPVP Arena — Plataforma Híbrida de Torneos

Ecosistema completo para la gestión y visualización de torneos gaming retro. El proyecto integra un cliente web interactivo para los usuarios junto a un potente panel de administración de escritorio desarrollado en JavaFX.

La persistencia y lógica de negocio se gestionan mediante una arquitectura relacional de base de datos conectada por JDBC, permitiendo la administración en tiempo real de usuarios, clasificaciones y emparejamientos.

Proyecto de Verano

BookStream — Red Social Literaria

Plataforma web estilo Goodreads para amantes de la lectura. El sistema permite a los usuarios registrar lecturas, puntuar libros, redactar reseñas y seguir la actividad literaria de otros perfiles en tiempo real.

Desarrollado con un backend robusto en Python y Django mediante una arquitectura de API REST, consumida de forma asíncrona por una SPA (Single Page Application) ágil e interactiva construida en React.
